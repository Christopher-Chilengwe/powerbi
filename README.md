## Rockview University KPI Reporting Template

A KPI dashboard template has been generated for Rockview University in `Rockview_University_KPI_Template.xlsx`.

### What is included

- `Data Entry` sheet with department-level KPIs for 21 areas.
- Pre-filled targets, definitions, frequencies, and dashboard formulas.
- `Summary Dashboard` sheet to track department performance.

### How to use

1. Open `Rockview_University_KPI_Template.xlsx`.
2. In the `Data Entry` sheet, enter monthly values for `Jan`, `Feb`, and `Mar`.
3. The `Q1 Avg` column calculates the average automatically.
4. The `Status` column shows `On Target` or `Below Target` based on the target and goal direction.
5. Use the `Summary Dashboard` sheet to review overall performance by department.

### Optional script

- `generate_kpi_excel.py` can regenerate the template if you need to update the KPI list.

### Notes

- Add responsible person names in the `Responsible` column.
- Use this file for monthly/quarterly submission to the M&E office.
